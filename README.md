
 # 💎 Jumka - E-Commerce Jewelry Website

A modern, responsive front-end web project showcasing a stylish **Jumka (earrings) e-commerce landing page** built using HTML and CSS.

This project demonstrates clean UI design, responsive layout, and product presentation suitable for an online jewelry store.

## 🚀 Features

- 🎯 Clean and elegant homepage design
- 🔍 Search bar for product discovery UI
- 🛍️ Product showcase section with hover effects
- 📱 Fully responsive design (mobile + desktop)
- 📖 About section describing brand identity
- 📞 Contact section for customer reach
- 🎨 Modern UI with hover animations and shadow effects

---

## 🛠️ Tech Stack

- HTML5
- CSS3 (Flexbox, Media Queries)
- Responsive Web Design
